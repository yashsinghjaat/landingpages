# Startup Landing Page Generator

A pure frontend project that allows users to generate a simple startup landing page preview and export it as a standalone HTML file. No backend or database required; everything runs in the browser with JavaScript.

## Features

- Input form for startup name, tagline, and any number of features (add more with + button)
- Theme selector (Light, Dark, Gradient, Minimal) plus custom background/text colors via color pickers
- Hero image URL support for background imagery
- Live preview updates as you type
- Export the generated landing page as an HTML file
- Copy HTML snippet to clipboard
- Responsive design with modern UI

## Usage

1. Open `index.html` in your browser.
2. Fill in the form on the left. The preview updates instantly.
3. Choose a theme from the dropdown.
4. Click *Download Landing Page* to save a complete HTML file.
5. Optionally, click *Copy HTML* to copy the preview markup.

## Deployment

You can host this project on GitHub Pages, Netlify, or Vercel. For GitHub Pages:

1. Push the `landing-page-generator` folder to a repository.
2. In repository settings, enable GitHub Pages and select the `main` branch.
3. Access the site at `https://<yourname>.github.io/<repo-name>/`.
